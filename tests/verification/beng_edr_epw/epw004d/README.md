# EP-W004d — Oriëntatie hoofdgevel naar Noord

EPW001 waarbij de gevel met ramen van **Zuid naar Noord** draait; alle overige
gevels draaien mee (gebouw roteert 180°). ISSO 54 v2.0, §2.2.3 deeltest d (p8).

## Delta

| Gevel | EPW001 | EPW004d |
|---|---|---|
| raamgevel | Zuid (180°) | Noord (0°) |
| tegenoverliggend | Noord (0°) | Zuid (180°) |
| oost | 90° | West (270°) |
| west | 270° | Oost (90°) |

Vlakoppervlakken en alle installaties ongewijzigd.

## Waarom deze variant

Zuivere oriëntatie-delta: isoleert de azimut-/zonwinst-tak (Q_sol) en de
TOjuli-berekening. Q_H;nd en TOjuli stijgen t.o.v. EPW001 door verlies van de
zuidelijke zonwinst — de exacte getallen staan in Bijlage 2.
