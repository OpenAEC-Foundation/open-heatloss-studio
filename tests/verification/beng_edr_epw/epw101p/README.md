# EP-W101p — Ventilatiesysteem D1 (geen WTW)

EPW001 met ventilatiesysteem **D1** (gebalanceerd, mechanische toe-/afvoer zónder
warmteterugwinning) i.p.v. het referentiesysteem **D2** (mét WTW). ISSO 54 v2.0,
§2.3.1 deeltest p (p20).

## Delta

| Kenmerk | EPW001 (D2) | EPW101p (D1) |
|---|---|---|
| WTW | ja (kunststof tegenstroom, bypass 1,0) | **geen** |
| Topologie | centraal | centraal |
| Luchtdichtheid kanalen | LUKA C | LUKA C |

## Waarom deze variant

Test de η_wtw-uitschakeling: EPW001 heeft WTW, D1 niet. De verwarmingsvraag
(Q_H;nd, EP1) stijgt door verlies van warmteterugwinning; de ventilatie-hulpenergie
(EV;ci) wijzigt. Raakt de ventilatie/infiltratie-tak (H_V). De exacte getallen
staan in Bijlage 2.
